# AI Virtual Assistant (MERN + Gemini API)


AI Virtual Assistant is a full-stack MERN application that allows users to interact with an intelligent chatbot powered by Google Gemini AI.  
It supports natural language conversation, task automation, and a modern UI built with React + Vite + Tailwind CSS.
